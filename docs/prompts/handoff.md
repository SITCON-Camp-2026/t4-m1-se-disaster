# Handoff Prompt

```text
請閱讀 README.md、docs/spec.md、docs/data-contract.md、docs/decisions.md、docs/handoff.md，以及目前 repo 的主要程式碼。

請協助整理 handoff 文件，讓另一組可以在有限時間內接手。

請補充：
1. 如何啟動專案
2. 主流程在哪些檔案
3. 資料入口在哪裡
4. 目前完成哪些 acceptance criteria
5. 已知問題與限制
6. 下一位接手者可以做的一個小任務
7. event injection 後有哪些資料或 schema 變更

限制：
- 不要隱藏未完成事項
- 不要宣稱這是真實救災系統
- 不要把不確定資訊寫成已確認事實
- 不要新增功能，只整理文件

請先輸出整理建議，等小組確認後再修改 docs/handoff.md。
```
